# Moonlight2

**Moonlight2**는 대량의 RAW 파일에서 초고속으로 썸네일을 추출하여 미리보기를 가능하게 해주는 데스크탑 애플리케이션입니다.  
다양한 기능들을 지속적으로 업데이트할 예정입니다

## 주요 기능
- **초고속 추출:** 수백 장의 RAW 파일에서도 빠르게 썸네일을 뽑아냅니다.
- **파일 정리 (File Organizer):** 촬영 날짜, 파일 확장자 등 다양한 규칙에 따라 파일을 자동으로 분류하고 정리합니다.
- **RAW-JPG 매칭 (RAW Matcher):** RAW 파일과 대응하는 JPG 파일의 유무를 확인하여 (확장자 무관) 특정파일을 복사합니다.
- **일괄 이름 변경 (Batch Rename):** 강력한 패턴 기능을 사용하여 대량의 파일 이름을 한 번에 규칙적으로 변경할 수 있습니다.
- **멀티 플랫폼 지원:** Windows와 macOS를 모두 지원합니다.
- 차후 다른 기능들 업데이트 예정

## ⚠️ 설치 전 확인해주세요! (중요)

Moonlight2는 개인 개발자가 만든 프로그램으로, 아직 공식 인증서(Code Signing)가 등록되지 않아 설치 시 보안 경고가 발생할 수 있습니다. 이는 프로그램의 문제가 아니니 안심하고 진행하셔도 됩니다.


### Windows 사용자
1. 실행 시 **"Windows의 PC 보호"** 창이 뜨면 좌측 상단의 **[추가 정보]**를 클릭합니다.
2. 우측 하단에 나타나는 **[실행]** 버튼을 눌러 설치를 진행해 주세요.


### macOS 사용자 가이드 (필독)
macOS의 보안 정책으로 인해 앱 실행이 차단될 수 있습니다. 이 경우 아래 방법 중 하나를 선택하여 실행해 주세요.

**방법 1: 터미널 명령어로 즉시 해결 (권장)**
앱을 응용 프로그램(Applications) 폴더로 옮긴 후, 터미널(Terminal.app)을 열어 아래 명령어를 복사해서 입력해 주세요. 이 명령어는 시스템의 실행 차단 속성을 제거해 줍니다.

```bash
sudo xattr -rd com.apple.quarantine /Applications/Moonlight2.app
```

터미널에 비밀번호를 입력하라고 나오면 맥 잠금해제 비밀번호를 넣고 엔터를 누릅니다.

비밀번호를 적을때 터미널에 변화가 없지만 그대로 입력하면 됩니다.
